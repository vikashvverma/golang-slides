# Slides for Introduction to Golang

## Running slides

Please make sure that $GOPATH/bin is included to your $PATH.

```bash
$ git clone https://github.com/vikashvverma/golang-slides.git
$ cd golang-slides
$ go get golang.org/x/tools/cmd/present
$ present
```
